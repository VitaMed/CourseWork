# CourseWork
This project is a food delivery site.
The main functions that are implemented in the project:
  * implementation of the catalog of goods: 
    * for each group of goods such characteristics as  img, ingredients, weight, cost are provided;
    * it is possible to order several units;
  * implementation of the ordering process;
  * adding an item to the cart;
  * removing an item from the cart;
  * acceptance of the order;
  * registration implementation;
  * implementation of the login.
